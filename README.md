# 🍔 Food Delivery App


## 📌 Project Overview

A professional food delivery application frontend inspired by platforms like Swiggy and Zomato.

The project provides a smooth food browsing experience with categories, restaurants, food cards, and interactive UI elements.

## ✨ Features

- Restaurant listing
- Food categories
- Search functionality
- Trending foods section
- Food cards
- Ratings and pricing
- Add to cart UI
- Wishlist UI
- Responsive design
- Horizontal scrolling sections
